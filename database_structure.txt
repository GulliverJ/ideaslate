DEVELOPER_ROLES
 - Used to see who's working which role on which projects
 - If user_id == NULL, it indicates that the associated role is available
   on the associated project

 +------------+---------+------+-----+---------+-------+
 | Field      | Type    | Null | Key | Default | Extra |
 +------------+---------+------+-----+---------+-------+

 | project_id | int(11) | NO   | MUL | NULL    |       |

 +------------+---------+------+-----+---------+-------+
